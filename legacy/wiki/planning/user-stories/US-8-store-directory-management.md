# US-8: Store Catalog & Configuration

> *"As a user, I want to manage where I shop and how those stores are laid out."*

## Goal
Provide a catalog view to manage stores and a configuration interface to customize store details and section layouts.

## Acceptance Criteria
- [ ] **Catalog View:** Lists all stores, grouped by household
- [ ] **Store Card:** Shows Name, Location, and "Start List" / "Edit" actions
- [ ] **Add Store:** Ability to create a new store within a household
- [ ] **Store Config:** Edit screen for Name, Location, and Image
- [ ] **Section Management:** Ability to add, remove, and reorder store sections (e.g., Produce, Dairy)
