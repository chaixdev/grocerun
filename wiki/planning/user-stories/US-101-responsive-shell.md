# US-101: Responsive Shell

> *"As a user, I want a responsive interface that works on mobile and desktop so that I can use the app anywhere."*

## Goal
Create the visual container for the application. It must be responsive, supporting a sidebar on desktop and a bottom navigation bar on mobile, ensuring a native-like experience.

## Acceptance Criteria
- [ ] **Responsive Layout**: Sidebar for Desktop (>768px), Bottom Nav for Mobile (<768px).
- [ ] **Navigation**: Users can switch between major views (Dashboard, Stores, Settings).
- [ ] **Theme Support**: Users can toggle between Light and Dark modes.
- [ ] **User Profile**: Users can see their current email and household name.
- [ ] **Logout**: Users can securely sign out.
