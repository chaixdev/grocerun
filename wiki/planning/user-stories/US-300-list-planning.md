# US-300: List Planning

> *"As a user, I want to create lists and add items quickly so that I don't forget anything."*

## Goal
The core "Planning Mode" where users build their shopping lists. This involves managing a catalog of items and adding them to specific lists.

## Acceptance Criteria
- [ ] **Catalog**: System remembers items I've bought before (Smart Autocomplete).
- [ ] **List Management**: Create, Rename, Delete shopping lists.
- [ ] **Item Entry**: Fast entry of items with quantity and units.
- [ ] **Categorization**: Items are automatically assigned to the correct Store Section.
