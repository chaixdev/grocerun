# User Stories

This directory contains consolidated user stories for Grocerun, updated with current implementation status and test coverage.

## Story Index

### ✅ Implemented
- [US-3: Household Collaboration](./US-03-household-collaboration.md) - Multi-user households with invitation system
- [US-8: Store Management](./US-08-store-management.md) - Store catalog and basic configuration ⚠️ *Sections pending*
- [US-10: Settings & Preferences](./US-10-settings-preferences.md) - Settings page with household management ⚠️ *Profile/theme pending*

### 🔲 Legacy Stories (Not Yet Migrated)
The following stories from the legacy wiki (`apps/web/wiki-legacy/planning/user-stories/`) have not been migrated as they represent future functionality:

- **US-1**: Autocomplete for Item Selection
- **US-2**: Common Items Catalog
- **US-4**: In-Trip Store Configuration (Deferred)
- **US-5**: Mobile Quantity Editing
- **US-6**: Responsive Shell
- **US-7**: Dashboard Experience
- **US-9**: Shopping State Machine (Planning/Shopping/Archived)

These will be migrated when implementation begins.

## Status Legend

- ✅ **IMPLEMENTED** - Feature complete with test coverage
- ⚠️ **PARTIALLY IMPLEMENTED** - Core functionality complete, some features pending
- 🔲 **PLANNED** - Design complete, implementation pending
- ❌ **DEPRECATED** - Story superseded or cancelled
