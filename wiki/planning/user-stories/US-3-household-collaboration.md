# US-3: Household Collaboration

> *"As a user, I want to have a common view on lists for my household."*

## Goal

## Acceptance Criteria
- [ ] Invite users to a household by email
- [ ] All household members see the same stores and lists
- [ ] Check-off updates are visible to all viewers within 5 seconds
