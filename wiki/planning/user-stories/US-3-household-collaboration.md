# US-3: Household Collaboration

> *"As a user, I want to have a common view on lists for my household."*

## Goal
Enable multiple household members to share stores and lists, with real-time updates during shopping.

## Acceptance Criteria
- [ ] **Generate Invite**: User can generate a single-use invitation token (valid 24h).
- [ ] **Share Invite**: User can copy the token to share via external channels.
- [ ] **Join Household**: User can enter a valid token to join a household.
- [ ] **Validation**: System prevents joining with expired or used tokens.
- [ ] **Visibility**: New members immediately see shared stores and lists.
