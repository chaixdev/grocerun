# GRO-310: Dashboard & Active Lists

**User Story**: [US-300: List Planning](../user-stories/US-300-list-planning.md)
**Status**: TODO

## Context
The home screen of the app.

## Requirements
1.  **Dashboard View**:
    -   Show "Active Lists" (Status = PLANNING or SHOPPING).
    -   Show "Recent Stores" (optional).
2.  **List Card**:
    -   Show Store Name.
    -   Show Item Count (e.g., "12 items").
    -   Show Status Badge.
    -   Click -> Go to Planning Mode (or Shopping Mode if active).

## Acceptance Criteria
- [ ] Dashboard displays active lists.
- [ ] Empty state when no lists exist.
