# GRO-330: Planning Mode UI

**User Story**: [US-300: List Planning](../user-stories/US-300-list-planning.md)
**Status**: TODO

## Context
The interface for adding items to a list.

## Requirements
1.  **Autocomplete Input**:
    -   Search `CatalogItem`s as user types.
    -   Create new `CatalogItem` if not found.
2.  **Item List**:
    -   Grouped by `Section`.
    -   Sorted by Section Order.
3.  **Item Controls**:
    -   Edit Quantity.
    -   Edit Unit.
    -   Remove Item.

## Acceptance Criteria
- [ ] User can add items.
- [ ] Items appear under correct sections.
- [ ] Autocomplete suggests existing items.
