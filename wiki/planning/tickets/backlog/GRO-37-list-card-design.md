# GRO-37: Design Active List Card Component

**Story**: "US-7: The Dashboard"
**Status**: Todo

## Context
The core element of the dashboard. Needs to show the status of a list at a glance.

## Requirements
1.  **Visuals**:
    -   Store Name (Title)
    -   Item Count / Status (Subtitle)
    -   "General List" variant (distinct icon/style)
2.  **States**:
    -   Planning (Default)
    -   Shopping (Active/Highlighted)
    -   Completed (Archived style)

## Acceptance Criteria
- [ ] Component renders with Store Name
- [ ] Component shows item count
- [ ] Visual distinction for "General List"
- [ ] Visual distinction for "Shopping Mode"
