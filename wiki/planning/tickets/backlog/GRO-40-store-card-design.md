# GRO-40: Design Store Card Component

**Story**: "US-8: Store Catalog & Configuration"
**Status**: Todo

## Context
The entry point for managing stores and starting lists.

## Requirements
1.  **Visuals**:
    -   Store Name
    -   Location (Subtitle)
2.  **Actions**:
    -   **Primary**: "List" button (Navigate to existing or create new).
    -   **Secondary**: "Edit" button (or menu icon) for configuration.

## Acceptance Criteria
- [ ] Component renders Store Name and Location
- [ ] "List" button is prominent
- [ ] "Edit" action is accessible but secondary
