## Context

## Requirements
1.  **List View**:
    -   Display all stores in the household.
    -   Card layout with Store Name.
2.  **Add Store**:
    -   Button to create a new store.
    -   Dialog to enter name.
    -   **Auto-create default sections** upon creation.

## Acceptance Criteria
- [ ] User can see list of stores.
- [ ] User can add a new store.
- [ ] New store appears immediately (Local-First).
