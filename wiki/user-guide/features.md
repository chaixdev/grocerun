# Grocerun Features

## Core Capabilities

### 🛒 Smart Shopping Lists
- Create lists organized by store
- Items auto-sort by store layout (produce → dairy → checkout)
- Real-time collaboration (multiple users shopping together)
- Quick-add from catalog (no retyping "organic milk" every week)

### 🏪 Store Layouts
- Customize sections per store (Trader Joe's ≠ Costco)
- Drag-and-drop section reordering
- Growing catalog learns where items live

### 👥 Household Sharing
- Invite family/roommates via simple codes (no email required)
- Everyone sees the same lists in real-time
- Per-user "who checked this off?" tracking

### 📱 Mobile-First Design
- Optimized for one-handed shopping
- Offline-capable (Phase 4: RxDB local-first)
- PWA installable (add to home screen)

### 🎨 Modern UI
- Dark mode (because grocery shopping at 9pm)
- Glassmorphism aesthetic
- Fast, responsive interactions

## Upcoming Features

### Phase 2: API Layer
- Backend separation for performance
- Foundation for offline sync

### Phase 3: Enhanced Features
- Voice input ("add milk")
- Smart suggestions (machine learning from shopping patterns)
- Recipe integration (add all ingredients at once)

### Phase 4: Local-First
- Full offline shopping
- Background sync when online
- Conflict resolution for multi-user edits
