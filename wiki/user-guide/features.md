# Features

Grocerun is a modern, real-time grocery list application.

## Core Features

- **Smart Catalog**: Items are automatically categorized.
- **Mobile First**: Designed to feel like a native app on your phone.
- **Offline Capable**: (Coming Soon) View your lists even without internet.

## User Interface

- **Dark Mode**: Easy on the eyes for late-night shopping.
- **Glassmorphism**: Premium visual aesthetic.
